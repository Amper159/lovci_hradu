# 🏰 Lovci Hradu - Průvodce českými hrady a zříceninami

Objevujte krásu českého dědictví! **Lovci Hradu** je webová aplikace, která vám pomáhá objevovat hrady, zříceniny a další historické památky po celé České republice. Ať už jste nadšenec pro historii nebo jen hledáte zajímavý cíl na svůj výlet, tady najdete všechno co potřebujete.

## 🎯 O projektu

Lovci Hradu je průvodce zaměřený na milovníky historie a aktivní turisty. Aplikace vám umožňuje:

- 🗺️ **Objevovat hrady a zříceniny** - Procházejte databází historických lokalit po celé ČR
- 📍 **Plánovat výlety** - Najděte nejbližší zajímavá místa pro vaši cestu
- 🏛️ **Poznat historii** - Zjistěte více o návštěvovaných stavbách a jejich historii
- 👥 **Sdílet zážitky** - Pokud je to podporováno, podělte se o své nápady a zkušenosti s ostatními

## 🚀 Jak začít

Aplikace je dostupná jako webová stránka - **není potřeba žádná instalace**. Jednoduše:

1. Otevřete aplikaci v prohlížeči
2. Procházejte dostupné hrady a zříceniny
3. Vyberte si váš příští cíl na výlet!

## 👥 Pro koho je určena?

Aplikace je ideální pro:
